# Images Directory

This directory contains all images used on the North Crescent Facility Solutions Inc website, organized by category.

## Directory Structure

```
images/
├── logo/
│   └── logo.png          # Company logo and branding
├── hero/
│   └── hero.jpg          # Hero section background image
├── residential/
│   └── residential.jpg   # Residential cleaning service
├── commercial/
│   └── commercial.jpg    # Commercial cleaning service
├── restaurant/
│   └── restaurant.jpg    # Restaurant cleaning service
├── postconstruction/
│   └── postconstruction.jpg  # Post-construction cleaning service
├── school/
│   └── school.jpg        # School cleaning service
└── team/
    └── team.jpg          # Team and company photos
```

## Adding New Images

Place new images in the appropriate subdirectory matching their service category. Use descriptive, lowercase filenames with hyphens for spaces (e.g., `residential-kitchen.jpg`).
